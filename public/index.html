Instructions

Overview

This API provides the ability to check whether a character string matches the name of an Amazon product category in the root category "Office Products" on Amazon's website. Requests are made via HTTP and responses are returned via HTTP in JSON format. Successful responses have a 200 Status Code.

Requests

To verify whether a category exists, make a HTTP GET request to the URL:

https://amaz-search-swenddaj-1018.uc.r.appspot.com/categories/search/ + [category name to search for]

(e.g., https://amaz-search-swenddaj-1018.uc.r.appspot.com/categories/search/paper)

To get a list of all the categories in the database, make a HTTP GET request to the URL:

https://amaz-search-swenddaj-1018.uc.r.appspot.com/categories/listCategories

Responses

Responses are JSON objects with the following key/value members:

{

"validCategory": bool,

"categoryName": string,

"categoryNumber": string,

"url": string,

"path": string

}
